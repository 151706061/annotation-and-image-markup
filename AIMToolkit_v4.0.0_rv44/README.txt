This is an official release of the AIM version 4.0.0.

It consists of a model, documentation and library.
It is the first major release of the AIM 4.0. The release
contains AIM 4.0 schemas, documentation and Windows and
Linux-compatible code.

There are three folders, which are doc, lib and source.

The doc folder consists of AIM-related documents and XML schemas.
*Note: user should use AIM_v4_rv44_XML.xsd for creating and
validating AIM XML documents.

The lib folder contains binaries for AIM XML library and a tool called ANIVATR
compiled for various platforms.

The source folder contains C++ source code for building AIM library and
ANNIVATR.
